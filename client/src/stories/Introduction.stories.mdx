# EASTON Component Library

This is a component library that I have built to use within person projects to ensure a consistent coding standards.

- Styling: [styled-components](https://styled-components.com/)
- Type Checking: [PropTypes](https://reactjs.org/docs/typechecking-with-proptypes.html)
- Testing: [React-Testing-Library](https://testing-library.com/docs/react-testing-library/intro/)
